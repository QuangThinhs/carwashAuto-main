# KẾ HOẠCH LÀM VIỆC NHÓM - DỰ ÁN AUTOWASH PRO

## 1. Các hạng mục công việc chính
* **Quản trị & Thiết kế hệ thống:**
  * Khởi tạo dự án, thiết lập cấu trúc Spring Boot và cấu hình Git.
  * Phân tích luồng nghiệp vụ, thiết kế CSDL (ERD) và đặc tả REST API.
* **Phát triển Frontend (Client):**
  * Xây dựng giao diện SPA bằng Next.js/React và Tailwind CSS.
  * Tích hợp Axios để giao tiếp dữ liệu với Backend qua JSON.
* **Phát triển Backend (Server):**
  * Xây dựng kiến trúc phân tầng (Controller - Service - Repository).
  * Cấu hình Spring Security, JWT và BCrypt cho bảo mật hệ thống.
  * Hiện thực hóa các API nghiệp vụ: Quản lý bãi rửa, Đặt lịch, Khuyến mãi & Tích điểm.
* **Kiểm thử & Hoàn thiện:**
  * Kiểm thử các luồng nghiệp vụ End-to-End.
  * Rà soát, fix bug và đóng gói tài liệu báo cáo dự án.

## 2. Tiến độ thực hiện (Timeline)
* **Tuần 1:** Khảo sát yêu cầu, thiết kế Database và cấu hình môi trường Git base.
* **Tuần 2:** Dựng khung Next.js (Frontend) và cấu hình Spring Boot (Backend).
* **Tuần 3:** Phát triển các tính năng cốt lõi (Auth JWT, Quản lý xe, Đặt lịch).
* **Tuần 4:** Hoàn thiện nghiệp vụ nâng cao (Khuyến mãi, Tích điểm, Dashboard Admin).
* **Tuần 5:** Kiểm thử tích hợp toàn bộ hệ thống, sửa lỗi và viết báo cáo.
